### 툴 기능
1. 게임의 난이도 조정을 위해 볼링 공과 핀의 수치를 조정해 시뮬레이션
2. 게임 내의 카메라 연출을 위해 카메라 위치, 회전 등 값을 기록

### 기술 스택
DirectX, MFC, C++, Visual Studio  

### 실행 환경
Windows
