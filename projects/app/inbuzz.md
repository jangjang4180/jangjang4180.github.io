인테리어 사업을 쉽고 간편하게!  
견적요청부터 계약, 결제까지 인테리어 업체와 자재·시공업체 간의 매칭과 편의를 위한 다양한 서비스를 인버즈에서 경험해 보세요!  

### 기술 스택
C#, ASP.NET Core, Dart, Flutter

### 실행 환경
Android  

### 담당 작업
* API 서버 개발
* Android 앱 개발
